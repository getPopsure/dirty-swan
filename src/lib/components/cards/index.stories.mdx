# Cards

Cards are interactive elements that can be used to display an action that a user can take.

# Card with left icon

| attribute | unit                                               | description                                                                                                                                  | default value | required |
| --------- | -------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- | ------------- | -------- |
| title     | string                                             | The title text that needs to be displayed                                                                                                    | n/a           | true     |
| titleSize | [Title Size (s,m,b)](#title-size-small-medium-big) | Size of the title                                                                                                                            | medium        | false    |
| leftIcon  | [Icon](#icon)                                      | Icon displayed on the left hand side of the card                                                                                             | n/a           | false    |
| rightIcon | [Icon](#icon)                                      | Icon displayed on the right hand side of the card                                                                                            | n/a           | false    |
| state     | [Card State](#card-state)                          | State that describe the interation with the card                                                                                             | actionable    | false    |
| onClick   | function                                           | Event handling, called when user click on the card                                                                                           | n/a           | false    |
| tabIndex  | function                                           | Indicates wether an element can be focused. More on [tabindex](https://developer.mozilla.org/en-US/docs/Web/HTML/Global_attributes/tabindex) | n/a           | false    |

# Models

## Title Size (small, medium, big)

The card title can be set with the following properties: `small`, `medium`, `big`.

## Title Size (medium, big)

The card title can be set with the following properties: `medium`, `big`.

## Card State

The card state is an string literral type with the following properties:

| state        | description                                              |
| ------------ | -------------------------------------------------------- |
| `actionable` | User can hover on the card, the cursor is set as pointer |
| `static`     | Card is displayed as is, no hovering.                    |
| `muted`      | Card is displayed faded, no hovering.                    |

## Icon

The icon object is defined with the following poperties

```typescript
export interface Icon {
  src: string; // source of the image
  alt: string; // alternate text if the image can't be displayed
}
```
