# Card with checkbox input

The card with checkbox input can be useful for adding call to actions.

| attribute   | unit                                            | description                                                | default value | required |
| ----------- | ----------------------------------------------- | ---------------------------------------------------------- | ------------- | -------- |
| id          | string                                          | The ID that will be passed to the input component          | n/a           | true     |
| title       | string                                          | The title text that needs to be displayed                  | n/a           | true     |
| onChange    | function                                        | The function that will run when the input value is changed | n/a           | true     |
| description | string                                          | An optional card description                               | n/a           | false    |
| icon        | [Icon](?path=/story/jsx-cards-intro--page#icon) | Icon displayed on the left side of the card                | n/a           | false    |
