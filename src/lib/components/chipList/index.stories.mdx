## ChipList

This component is a list of Chip components.

The component takes in an array of options as `Option` type (see models for Option interface).

Each Chip can be removed by clicking on the X button with the provided `onRemove` function.

Images can shown on the chip, if provided in the `values` array (optional)

## Example

<Preview>
  <ChipList
    values={[
      { value: 'Private Health' },
      { value: 'Public Health' },
      { value: 'Dental' },
      { value: 'Legal' },
    ]}
    onRemove={() => {}}
  />
</Preview>
