## AutoSuggestInput

This component allows quick search via the input field to find an option for selection.

## Arguments

| Argument                       | Type     | Description                                                                               |
| ------------------------------ | -------- | ----------------------------------------------------------------------------------------- |
| `currentOption`                | string   | Current input of the component by user                                                    |
| `suggestions`                  | `Option` | List of suggestions that should be displayed to the respective input                      |
| `onChange`                     | function | Function that changes the value of current input                                          |
| `handleSuggestionSelected`     | function | Functon that runs when a suggestion is selected from the dropdown                         |
| `handleSuggestionFetchRequest` | function | Function that allows control of which suggestions should be displayed                     |
| `handleSuggestionClearRequest` | function | Function that runs when suggestions are cleared (eg. input removal, selecting suggestion) |
| `placeholder`                  | string   | Placeholder for DirtySwan Input component                                                 |
| `className` (optional)         | string   | Class name for the most parent element                                                    |
