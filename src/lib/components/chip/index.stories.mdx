## Chip

This component is Chip component that displays an image, selection value, and a remove button.

The component takes in an argument `value` as `Option` type (see models for Option interface).

Chip can be removed by clicking on the X button with the provided `onRemove` function.

Images can shown on the chip, if provided in the `value` prop (optional)
