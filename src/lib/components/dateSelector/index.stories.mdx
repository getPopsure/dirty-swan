# Date selector

Date selector are user interface elements which allow user to select a date in the `YYYY-MM-DD` format.

## Default example

<Preview>
  <DateSelector
    yearBoundaries={{ min: 1990, max: 2020 }}
    onChange={(date) => {
      console.log(date);
    }}
  />
</Preview>
