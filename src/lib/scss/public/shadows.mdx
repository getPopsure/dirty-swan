# Box Shadow

Dirty swan provides you with a set of box shadow classes and variables.

## Usage

### Class name

| Class    | Value                                              |
| -------- | ------------------------------------------------------------- |
| `.bs-xs` | `box-shadow: 0 2px 12px rgba($ds-grey-900, 0.03)`             |
| `.bs-sm` | `box-shadow: 0 2px 20px 0 rgba($ds-grey-900, 0.04)`           |
| `.bs-md` | `box-shadow: 0 2px 28px rgba($ds-grey-900, 0.08)`             |
| `.bs-lg` | `box-shadow: 0 2px 32px rgba($ds-grey-900, 0.1)`              |
| `.bs-xl` | `box-shadow: 0 2px 50px rgba($ds-grey-900, 0.18)`             |

### Import in code

```sass
// demo.scss
@import '@popsure/dirty-swan/dist/shadows';

.element {
  box-shadow: $bs-md;
}
```
