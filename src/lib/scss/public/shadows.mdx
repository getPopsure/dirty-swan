# Box Shadow

Dirty swan provides you with a set of box shadow classes and variables.

## Usage

### Class name

| Class    | Value                                              |
| -------- | -------------------------------------------------- |
| `.bs-xs` | `box-shadow: 0px 4px 6px rgba(97, 96, 162, 0.01)`  |
| `.bs-sm` | `box-shadow: 0px 2px 28px rgba(97, 96, 162, 0.1)`  |
| `.bs-md` | `box-shadow: 0px 2px 28px rgba(97, 96, 162, 0.12)` |
| `.bs-lg` | `box-shadow: 0px 2px 28px rgba(97, 96, 162, 0.15)` |

### Import in code

```sass
// demo.scss
@import '@popsure/dirty-swan/dist/shadows';

.element {
  box-shadow: $bs-md;
}
```

## Table

| Variable | Value                                  |
