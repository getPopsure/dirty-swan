# Font weight

For enhanced reusability weâ€™re providing you with a set of font weight. Please don't hardcode any font weight value and use this instead.

## Usage

### Class name

| Class        | Output             |
| ------------ | ------------------ |
| `.fw-normal` | `font-weight: 400` |
| `.fw-bold`   | `font-weight: 700` |

### Import in code

```sass
// demo.scss
@import '@popsure/dirty-swan/dist/font-weight';

.element {
  font-weight: $p-font-weight-bold;
}
```

## Table

| Variable                | Value | Name   |
| ----------------------- | ----- | ------ |
| `$p-font-weight-normal` | 400   | normal |
| `$p-font-weight-bold`   | 600   | bold   |
