# Cursors

Utilities for controlling the cursor style when hovering over an element.

## Classes

| Class            | Output                 |
| ---------------- | ---------------------- |
| `.c-auto`        | `cursor: auto;`        |
| `.c-default`     | `cursor: default;`     |
| `.c-pointer`     | `cursor: pointer;`     |
| `.c-text`        | `cursor: text;`        |
| `.c-wait`        | `cursor: wait;`        |
| `.c-move`        | `cursor: move;`        |
| `.c-not-allowed` | `cursor: not-allowed;` |
| `.c-help`        | `cursor: help;`        |
