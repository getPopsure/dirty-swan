# Animation

Dirty Swan provides class names for animation effects to quickly improve the UI look.

## Animation types

### Appear In

Giving class name `ds-appear-animation` to a component allows it to appear on screen with a slight delay

<Preview>
  <h1 class="ds-appear-animation">Dirty Swan makes my life easier!</h1>
</Preview>

## Animation delays

Simply delays the animation with class names

| Class        | CSS       |
| ------------ | --------- |
| `ds-animation-delay-1`   | animation-delay: 0.1s; | 
| `ds-animation-delay-2`   | animation-delay: 0.2s; | 
| `ds-animation-delay-3`   | animation-delay: 0.3s; | 
| `ds-animation-delay-4`   | animation-delay: 0.4s; | 
| `ds-animation-delay-5`   | animation-delay: 0.5s; | 

### Example

<Preview>
  <h1 class="ds-appear-animation ds-animation-delay-3">Dirty Swan makes my life easier!</h1>
</Preview>
