# Width and height

Dirty swan provides atomic sizing classes for widths and heights.

## Static widths

### Static widths classes

| Class   | Output        |
| ------- | ------------- |
| `.ws1`  | width: 60px;  |
| `.ws2`  | width: 136px; |
| `.ws3`  | width: 212px; |
| `.ws4`  | width: 288px; |
| `.ws5`  | width: 364px; |
| `.ws6`  | width: 440px; |
| `.ws7`  | width: 516px; |
| `.ws8`  | width: 592px; |
| `.ws9`  | width: 668px; |
| `.ws10` | width: 744px; |
| `.ws11` | width: 820px; |
| `.ws12` | width: 896px; |

### Static width examples

```html
<div class="ws1">…</div>
<div class="ws2">…</div>
<div class="ws3">…</div>
<div class="ws4">…</div>
<div class="ws5">…</div>
<div class="ws6">…</div>
<div class="ws7">…</div>
<div class="ws8">…</div>
<div class="ws9">…</div>
<div class="ws10">…</div>
<div class="ws11">…</div>
<div class="ws12">…</div>
```

## Fuild width widths

### Fuild widths classes

| Class     | Output        |
| --------- | ------------- |
| `.w-auto` | width: auto;  |
| `.w0`     | width: 0;     |
| `.w10`    | width: 10%;   |
| `.w20`    | width: 20%;   |
| `.w30`    | width: 30%;   |
| `.w40`    | width: 40%;   |
| `.w50`    | width: 50%;   |
| `.w60`    | width: 60%;   |
| `.w70`    | width: 70%;   |
| `.w80`    | width: 80%;   |
| `.w90`    | width: 90%;   |
| `.w100`   | width: 1000%; |
